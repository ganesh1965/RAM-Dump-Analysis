
Commands Used & Their Purpose

1 List Running Processes

vol.exe -f dump.raw windows.pslist

Use: Shows all processes active at the time of RAM capture.


2 List Network Connections

vol.exe -f dump.raw windows.netscan

Use: Displays all active TCP/UDP connections from the memory dump.


3 View Executed Commands (Command History)

vol.exe -f dump.raw windows.cmdscan

Use: Retrieves commands that were executed in CMD before the dump.


4 Get Basic System Information

vol.exe -f dump.raw windows.info

Use: Confirms the dump file details (OS, version, memory layer).
